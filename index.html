Project Info
  • Project Name: jsf-extensions
  • Latest Release: 0.1alpha1
  • Latest Release Status: Early Access
  • Project Type: Web Application Framework Extension

jsf-extensions is a collection of mostly non-UI components built on top of the JSF specification, leveraging the extreme extensibility of the spec to enhance the functionality of the implementation.

Download and Installation

Early Access Releases

0.1alpha1 15 December 2005

Download and unpack the zip. It contains a sample war that is ready to deploy on build 30 or later of glassfish. If you want to get right to using it, place the jars in the jars directory in your WEB-INF/lib and proceed to the documentation.

Support

JavaServer Faces is a technology, not a product. Paid support may be available from the Application Server vendor that includes JavaServer Faces technology, for example, the Sun Java System Application Server.

The javaserverfaces project here on java.net is a community-supported open development product. The best venue for getting your questions answered is the Java Developer Connection Forum about JavaServer Faces Technology.
