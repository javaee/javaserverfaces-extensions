Project Dynamic Faces Samples

This directory contains sample applications illustrating the usage of Project Dynamic Faces.

How to build the samples

Building the samples is easy! You need Apache Ant. If you are running this installation inside of Sun's Java System Application Server you already have ant in the form of SUNWappserver/bin/asant, where SUNWappserver is the installation directory for the Application Server. Otherwise, you'll have to install ant from The apache web site.

Once you have ant installed, examine the build.properties file in this directory to ensure that the javaee.jar property points to a valid javaee.jar file. If necessary, modify the file accordingly.

Finally, simply type "ant" at a command prompt in this directory to build the samples.

How to install the samples

Once the samples have built successfully, for each sample you will see a "build" directory. In that directory will be a WAR file, ready to deploy on your Application Server, and an exploded WAR directory which can be deployed with the "deploydir" feature of the Application Server.

How to run the samples

Just visit http://server:port/context-path/ in a Firefox web browser, where server is the host on which the Application Server is installed, port is the port on which the Application Server is running on that host, and context-path is the part of the filename before the ".war" in the file in the build directory for that sample. Each sample has its own README.html file in its sample directory.
